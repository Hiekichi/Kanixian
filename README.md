# Kanixian

## 概要
- プログラミング学習用に作成したゲームプログラムです。
- ナニカ問題がありましたら対応させていただきたいと思います。

![スクリーンショット](https://cdn-ak.f.st-hatena.com/images/fotolife/H/Hiesuke/20240521/20240521231058.png)

## 遊び方など
- ゲームパッドかキーボードで操作できます。
- 飛んでくる敵を打つ！だけです。
- 飛行中の敵は高得点です。
- 編隊飛行はしてくれません（ゴメンナサイ）
